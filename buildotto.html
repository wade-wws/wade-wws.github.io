Build Otto




Highlights






A back-end service to process builder blueprint into home automation bundle for quoting to clients




Description

Build Otto allows for builders to login and upload thier buildprints to have them processed into bundles for quoting back to their clients.


Key features

  • Uses buildprints to plan and quote a home automation system bundle
  • Creates quotes from blueprints for customization and then for ordering
  • Orders are converted over to home otto sites and hardware is pre-programmed
  • Custom Designed system for reading and processing buildprints
  • Administrative account management
  • Integrated with Stripe for payment processing
  • Integrated with Slack for customer relations, and system processing
  • Many other proprietary systems for a full end-to-end solution

Quotes, Orders, and Blueprints

  • Easily manages client quotes and orders per account
  • Uploads blueprint documents with AWS S3 storage

Custom Home automation quoting interface

  • Add Floors, Rooms, Lights, and other hardware in a custom designed interface
  • Guided process to completion or a fast one-click quoting

Blueprint system viewer

  • Full blueprint viewer in a custom web based system
  • Zoom in and view each floor and rooms details

Quote and order cart

  • Quoting and Ordering cart to process into shipped bundle

Custom blueprint editor

  • Web based CAD like system to draw out and connect
