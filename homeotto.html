Home Otto

A complete end-to-end home automation kit for new home owners.




Description

Home Otto is a a complete package of home automation products for builders to pre-install into new home builds.


Key features

  • Mobile app for home owners with thier new home
  • Just one account for all your automation devices
  • Zero setup, ready to use minute one
  • Uses geofences to know when your home or away
  • Set anything with you voice with Amazon Alexa
  • Real-time changes, no delays
  • Built for iOS, Android, and Amazon Alexa

Mobile / Voice app

  • Available on iOS, Android, and Amazon Alexa
  • Hybrid app that updates without new store releases
  • Real-time changes from app to home
  • Inegration with location and notification services
