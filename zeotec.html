Lab testing for Chemical and Non-Chemical services




Description

Zeotec app allows for in-the-field service technicians to login and manage ice rink chillers and equipment lab testing.


Key features

  • Mobile access for service technicians
  • Map overview to quickly identify issues
  • Asset management and contracts
  • Real-time lab results, for fast reaction times
  • Integrated emailing for client approvals
  • Print, Download, and Email lab reports

Map Overview

  • Monitor all locations in a single view
  • Fully interactive map to pull up details on selected location
  • Indicators to monitor for issues
  • Add notes inline to pass back inportant details

Search / Bookmarked locations

  • Each user has stored locations for quick access
  • Search by location details or other points of reference

Asset Management

  • Manage each system independantly for age
  • Monitor contract details for out-of-service

Lab Testing

  • Lab testing per system
  • Generated Label for easy sample labeling
